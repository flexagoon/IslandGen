# IslandGen

![Island](https://fxgn.dev/blog/making-islands/full.png)

A simple experiment in procedural generation, this script generates random islands like the one above.

## Details

Full information about this project can be found in [this blog post](http://fxgn.dev/blog/making-islands).

## Running

If you have [just](https://just.systems) installed:

```sh
just run
```

Otherwise:

```
julia --project=. src/IslandGen.jl
```
